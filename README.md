Drug Response Analysis Using SAS

This project analyzes patient clinical parameters to determine factors influencing drug prescription patterns.

Tools Used
- SAS OnDemand for Academics
- SAS Studio

 Statistical Methods
- Descriptive statistics
- Chi-square test
- Boxplot visualization

 Key Findings
- Blood pressure significantly affects drug prescription
- Cholesterol levels influence drug selection
- Na/K ratio strongly predicts DrugY
- Gender has no significant effect

Dataset
Drug response dataset with 200 patient records.
